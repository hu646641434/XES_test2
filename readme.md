## 安装
### Installation with CocoaPods

1.在 `Podfile` 中添加 `pod 'Test2Project'`。

2.执行 `pod install` 或 `pod update`。

3.导入 `"Test2Project.h"`。

### 手动安装
1.下载Test2Project文件夹内的所有内容。

2.将Test2Project内的源文件添加（拖放）到你的工程。

3.导入Test2Project.h。
